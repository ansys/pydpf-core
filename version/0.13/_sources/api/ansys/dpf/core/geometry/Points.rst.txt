





:class:`Points`
===============

.. py:class:: ansys.dpf.core.geometry.Points(coordinates, server=None)

   Collection of DPF points.

   This can be a single point or a cloud of points.

   :param coordinates: Coordinates of the points in a 3D space.
   :type coordinates: array, list, Field
   :param server: Server with the channel connected to the remote or local instance. The
                  default is ``None``, in which case an attempt is made to use the global
                  server.
   :type server: :class:`ansys.dpf.core.server`, optional

   .. rubric:: Examples

   Create Points object with two points, print object information and plot.

   >>> from ansys.dpf.core.geometry import Points
   >>> points = Points([[0, 0, 0], [1, 0, 0]])
   >>> print(points)
   DPF Points object:
   Number of points: 2
   Coordinates:
     [0. 0. 0.]
     [1. 0. 0.]
   >>> points.plot()
