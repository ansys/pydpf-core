.. py:property:: input_string
   :type: ansys.dpf.core.inputs.Input


   Allows to connect input_string input to the operator.

   ex: 'double:1.0', 'int:1', 'vector<double>:1.0;1.0'

   :returns: An Input instance for this pin.
   :rtype: input
