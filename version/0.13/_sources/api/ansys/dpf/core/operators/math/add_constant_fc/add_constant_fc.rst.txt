





:class:`add_constant_fc`
========================

.. py:class:: ansys.dpf.core.operators.math.add_constant_fc.add_constant_fc(fields_container=None, weights=None, config=None, server=None, ponderation=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Computes the sum of a field (in 0) and a scalar (in 1).


   :param fields_container: field or fields container with only one field is expected
   :type fields_container: FieldsContainer
   :param weights: double or vector of double
   :type weights: float

   :returns: **fields_container**
   :rtype: FieldsContainer

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.math.add_constant_fc()

   >>> # Make input connections
   >>> my_fields_container = dpf.FieldsContainer()
   >>> op.inputs.fields_container.connect(my_fields_container)
   >>> my_weights = float()
   >>> op.inputs.weights.connect(my_weights)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.math.add_constant_fc(
   ...     fields_container=my_fields_container,
   ...     weights=my_weights,
   ... )

   >>> # Get output data
   >>> result_fields_container = op.outputs.fields_container()
