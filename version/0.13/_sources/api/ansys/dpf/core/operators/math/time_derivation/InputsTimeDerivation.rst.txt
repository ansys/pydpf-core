





:class:`InputsTimeDerivation`
=============================

.. py:class:: ansys.dpf.core.operators.math.time_derivation.InputsTimeDerivation(op: ansys.dpf.core.dpf_operator.Operator)

   Intermediate class used to connect user inputs to
   time_derivation operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.math.time_derivation()
   >>> my_field = dpf.Field()
   >>> op.inputs.field.connect(my_field)
   >>> my_spline_fitting = bool()
   >>> op.inputs.spline_fitting.connect(my_spline_fitting)
