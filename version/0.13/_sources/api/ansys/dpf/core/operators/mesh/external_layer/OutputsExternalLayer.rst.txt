





:class:`OutputsExternalLayer`
=============================

.. py:class:: ansys.dpf.core.operators.mesh.external_layer.OutputsExternalLayer(op: ansys.dpf.core.dpf_operator.Operator)

   Bases: :py:obj:`ansys.dpf.core.outputs._Outputs`


   Intermediate class used to get outputs from
   external_layer operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.mesh.external_layer()
   >>> # Connect inputs : op.inputs. ...
   >>> result_mesh = op.outputs.mesh()
   >>> result_nodes_mesh_scoping = op.outputs.nodes_mesh_scoping()
   >>> result_elements_mesh_scoping = op.outputs.elements_mesh_scoping()
