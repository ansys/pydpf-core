





:class:`InputsMakePlaneLevelset`
================================


.. py:class:: ansys.dpf.core.operators.mesh.make_plane_levelset.InputsMakePlaneLevelset(op: ansys.dpf.core.dpf_operator.Operator)

   Bases: :py:obj:`ansys.dpf.core.inputs._Inputs`


   Intermediate class used to connect user inputs to
   make_plane_levelset operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.mesh.make_plane_levelset()
   >>> my_coordinates = dpf.MeshedRegion()
   >>> op.inputs.coordinates.connect(my_coordinates)
   >>> my_normal = dpf.Field()
   >>> op.inputs.normal.connect(my_normal)
   >>> my_origin = dpf.Field()
   >>> op.inputs.origin.connect(my_origin)
