





:class:`InputsNodalToGlobal`
============================


.. py:class:: ansys.dpf.core.operators.result.nodal_to_global.InputsNodalToGlobal(op: ansys.dpf.core.dpf_operator.Operator)

   Bases: :py:obj:`ansys.dpf.core.inputs._Inputs`


   Intermediate class used to connect user inputs to
   nodal_to_global operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.result.nodal_to_global()
   >>> my_fieldA = dpf.Field()
   >>> op.inputs.fieldA.connect(my_fieldA)
   >>> my_fieldB = dpf.Field()
   >>> op.inputs.fieldB.connect(my_fieldB)
   >>> my_inverse_rotation = bool()
   >>> op.inputs.inverse_rotation.connect(my_inverse_rotation)

.. py:property:: fieldA
   :type: ansys.dpf.core.inputs.Input[ansys.dpf.core.field.Field]


   Allows to connect fieldA input to the operator.

   Vector or tensor field that must be rotated, expressed in nodal coordinate system.

   :returns: An Input instance for this pin.
   :rtype: input

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.result.nodal_to_global()

   >>> op.inputs.fieldA(my_fieldA)

.. py:property:: fieldB
   :type: ansys.dpf.core.inputs.Input[ansys.dpf.core.field.Field]


   Allows to connect fieldB input to the operator.

   Nodal euler angles defined from a result file. Those must be the rotations from Nodal to Global.

   :returns: An Input instance for this pin.
   :rtype: input

.. py:property:: inverse_rotation
   :type: ansys.dpf.core.inputs.Input[bool]


   Allows to connect inverse_rotation input to the operator.

   If true, we apply the inverse rotation (default is false).

   :returns: An Input instance for this pin.
   :rtype: input

   .. rubric:: Examples

   >>> op = dpf.operators.result.nodal_to_global()

   >>> # or
   >>> op.inputs.inverse_rotation(my_inverse_rotation)
