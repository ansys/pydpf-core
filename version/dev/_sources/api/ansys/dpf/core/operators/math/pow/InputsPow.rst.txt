





:class:`InputsPow`
==================


.. py:class:: ansys.dpf.core.operators.math.pow.InputsPow(op: ansys.dpf.core.dpf_operator.Operator)

   Bases: :py:obj:`ansys.dpf.core.inputs._Inputs`


   Intermediate class used to connect user inputs to
   pow operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.math.pow()
   >>> my_field = dpf.Field()
   >>> op.inputs.field.connect(my_field)
   >>> my_factor = float()
   >>> op.inputs.factor.connect(my_factor)
   >>> my_division_by_zero_value = float()
   >>> op.inputs.division_by_zero_value.connect(my_division_by_zero_value)
