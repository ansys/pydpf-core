   Intermediate class used to connect user inputs to
   pow operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.math.pow()
   >>> my_field = dpf.Field()
   >>> op.inputs.field.connect(my_field)
   >>> my_factor = float()
   >>> op.inputs.factor.connect(my_factor)
