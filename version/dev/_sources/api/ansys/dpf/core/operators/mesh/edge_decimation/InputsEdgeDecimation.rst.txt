





:class:`InputsEdgeDecimation`
=============================


.. py:class:: ansys.dpf.core.operators.mesh.edge_decimation.InputsEdgeDecimation(op: ansys.dpf.core.dpf_operator.Operator)

   Bases: :py:obj:`ansys.dpf.core.inputs._Inputs`


   Intermediate class used to connect user inputs to
   edge_decimation operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.mesh.edge_decimation()
   >>> my_mesh = dpf.MeshedRegion()
   >>> op.inputs.mesh.connect(my_mesh)
   >>> my_angular_limit_in_radian = float()
   >>> op.inputs.angular_limit_in_radian.connect(my_angular_limit_in_radian)
