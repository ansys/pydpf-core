





:class:`identical_scopings`
===========================

.. py:class:: ansys.dpf.core.operators.logic.identical_scopings.identical_scopings(scopingA=None, scopingB=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Check if two scopings are identical.

   :param scopingA:
   :type scopingA: Scoping
   :param scopingB:
   :type scopingB: Scoping

   :returns: * **boolean** (*bool*) -- Bool (true if identical...)
             * **message** (*str*)

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.logic.identical_scopings()

   >>> # Make input connections
   >>> my_scopingA = dpf.Scoping()
   >>> op.inputs.scopingA.connect(my_scopingA)
   >>> my_scopingB = dpf.Scoping()
   >>> op.inputs.scopingB.connect(my_scopingB)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.logic.identical_scopings(
   ...     scopingA=my_scopingA,
   ...     scopingB=my_scopingB,
   ... )

   >>> # Get output data
   >>> result_boolean = op.outputs.boolean()
   >>> result_message = op.outputs.message()
