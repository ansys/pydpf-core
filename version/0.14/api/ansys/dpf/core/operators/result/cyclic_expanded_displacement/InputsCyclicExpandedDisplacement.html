InputsCyclicExpandedDisplacement#

class ansys.dpf.core.operators.result.cyclic_expanded_displacement.InputsCyclicExpandedDisplacement(op: ansys.dpf.core.dpf_operator.Operator)#

Bases: ansys.dpf.core.inputs._Inputs

Intermediate class used to connect user inputs to cyclic_expanded_displacement operator.

Examples

>>> from ansys.dpf import core as dpf
>>> op = dpf.operators.result.cyclic_expanded_displacement()

>>> my_mesh_scoping = dpf.ScopingsContainer()
