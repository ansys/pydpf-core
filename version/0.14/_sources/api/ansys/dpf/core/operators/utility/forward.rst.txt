ansys.dpf.core.operators.utility.forward
========================================

.. py:class:: ansys.dpf.core.operators.utility.forward(any=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Return all the inputs as outputs.


   :param any: any type of input
   :type any: Any

   :returns: **any** -- same types as inputs
   :rtype: Any

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.utility.forward()

   >>> # Make input connections
   >>> my_any = dpf.Any()
   >>> op.inputs.any.connect(my_any)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.utility.forward(
   ...     any=my_any,
   ... )

   >>> # Get output data
   >>> result_any = op.outputs.any()
