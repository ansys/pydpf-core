ansys.dpf.core.operators.geo.normals
====================================

.. py:class:: ansys.dpf.core.operators.geo.normals(mesh=None, mesh_scoping=None, field=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   compute the normals at the given nodes or element scoping based on the
   given mesh (first version, the element normal is only handled on the
   shell elements)


   :param mesh:
   :type mesh: MeshedRegion, optional
   :param mesh_scoping:
   :type mesh_scoping: Scoping, optional
   :param field:
   :type field: Field, optional

   :returns: **field**
   :rtype: Field

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.geo.normals()

   >>> # Make input connections
   >>> my_mesh = dpf.MeshedRegion()
   >>> op.inputs.mesh.connect(my_mesh)
   >>> my_mesh_scoping = dpf.Scoping()
   >>> op.inputs.mesh_scoping.connect(my_mesh_scoping)
   >>> my_field = dpf.Field()
   >>> op.inputs.field.connect(my_field)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.geo.normals(
   ...     mesh=my_mesh,
   ...     mesh_scoping=my_mesh_scoping,
   ...     field=my_field,
   ... )

   >>> # Get output data
   >>> result_field = op.outputs.field()
