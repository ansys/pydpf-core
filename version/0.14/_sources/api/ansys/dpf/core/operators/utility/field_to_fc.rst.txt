ansys.dpf.core.operators.utility.field_to_fc
============================================

.. py:class:: ansys.dpf.core.operators.utility.field_to_fc(field=None, label=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Creates a fields container containing the field provided on pin 0.


   :param field: If a fields container is set in input, it is passed on as an output with the additional label space (if any).
   :type field: Field or FieldsContainer
   :param label: Sets a label space.
   :type label: dict

   :returns: **fields_container**
   :rtype: FieldsContainer

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.utility.field_to_fc()

   >>> # Make input connections
   >>> my_field = dpf.Field()
   >>> op.inputs.field.connect(my_field)
   >>> my_label = dict()
   >>> op.inputs.label.connect(my_label)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.utility.field_to_fc(
   ...     field=my_field,
   ...     label=my_label,
   ... )

   >>> # Get output data
   >>> result_fields_container = op.outputs.fields_container()
