ansys.dpf.core.operators.utility.assemble_scalars_to_vectors
============================================================

.. py:class:: ansys.dpf.core.operators.utility.assemble_scalars_to_vectors(x=None, y=None, z=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Takes three scalar fields and assembles them as a 3D vector field.


   :param x:
   :type x: Field, optional
   :param y:
   :type y: Field, optional
   :param z:
   :type z: Field, optional

   :returns: **field**
   :rtype: Field

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.utility.assemble_scalars_to_vectors()

   >>> # Make input connections
   >>> my_x = dpf.Field()
   >>> op.inputs.x.connect(my_x)
   >>> my_y = dpf.Field()
   >>> op.inputs.y.connect(my_y)
   >>> my_z = dpf.Field()
   >>> op.inputs.z.connect(my_z)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.utility.assemble_scalars_to_vectors(
   ...     x=my_x,
   ...     y=my_y,
   ...     z=my_z,
   ... )

   >>> # Get output data
   >>> result_field = op.outputs.field()
