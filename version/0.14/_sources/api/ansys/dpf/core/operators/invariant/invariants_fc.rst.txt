ansys.dpf.core.operators.invariant.invariants_fc
================================================

.. py:class:: ansys.dpf.core.operators.invariant.invariants_fc(fields_container=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Computes the element-wise invariants of all the tensor fields of a
   fields container.


   :param fields_container:
   :type fields_container: FieldsContainer

   :returns: * **fields_int** (*FieldsContainer*) -- stress intensity field
             * **fields_eqv** (*FieldsContainer*) -- stress equivalent intensity
             * **fields_max_shear** (*FieldsContainer*) -- max shear stress field

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.invariant.invariants_fc()

   >>> # Make input connections
   >>> my_fields_container = dpf.FieldsContainer()
   >>> op.inputs.fields_container.connect(my_fields_container)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.invariant.invariants_fc(
   ...     fields_container=my_fields_container,
   ... )

   >>> # Get output data
   >>> result_fields_int = op.outputs.fields_int()
   >>> result_fields_eqv = op.outputs.fields_eqv()
   >>> result_fields_max_shear = op.outputs.fields_max_shear()
