ansys.dpf.core.operators.utility.merge_materials
================================================

.. py:class:: ansys.dpf.core.operators.utility.merge_materials(materials1=None, materials2=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Assembles a set of materials into a unique one.


   :param materials1: A vector of materials to merge or materials from pin 0 to ...
   :type materials1: Materials
   :param materials2: A vector of materials to merge or materials from pin 0 to ...
   :type materials2: Materials

   :returns: **merged_materials**
   :rtype: Materials

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.utility.merge_materials()

   >>> # Make input connections
   >>> my_materials1 = dpf.Materials()
   >>> op.inputs.materials1.connect(my_materials1)
   >>> my_materials2 = dpf.Materials()
   >>> op.inputs.materials2.connect(my_materials2)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.utility.merge_materials(
   ...     materials1=my_materials1,
   ...     materials2=my_materials2,
   ... )

   >>> # Get output data
   >>> result_merged_materials = op.outputs.merged_materials()
