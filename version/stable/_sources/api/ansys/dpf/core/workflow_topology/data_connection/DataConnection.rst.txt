





:class:`DataConnection`
=======================

.. py:class:: ansys.dpf.core.workflow_topology.data_connection.DataConnection(container: ansys.dpf.core.generic_data_container.GenericDataContainer)

   Bases: :py:obj:`ansys.dpf.core.custom_container_base.CustomContainerBase`


   Represents a connection between a data and an operator in a workflow.

   This class provides access to the source data and target operator, as well as its pin ID.
