





:class:`merge_collections`
==========================

.. py:class:: ansys.dpf.core.operators.utility.merge_collections.merge_collections(collections1=None, collections2=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Merges a set of collections into a unique one.


   :param collections1: a vector of collections to merge or collections from pin 0 to ...
   :type collections1: AnyCollection
   :param collections2: a vector of collections to merge or collections from pin 0 to ...
   :type collections2: AnyCollection

   :returns: **merged_collections**
   :rtype: AnyCollection

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.utility.merge_collections()

   >>> # Make input connections
   >>> my_collections1 = dpf.AnyCollection()
   >>> op.inputs.collections1.connect(my_collections1)
   >>> my_collections2 = dpf.AnyCollection()
   >>> op.inputs.collections2.connect(my_collections2)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.utility.merge_collections(
   ...     collections1=my_collections1,
   ...     collections2=my_collections2,
   ... )

   >>> # Get output data
   >>> result_merged_collections = op.outputs.merged_collections()
