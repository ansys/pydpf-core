Property detail
---------------

.. py:property:: output_path
   :type: ansys.dpf.core.inputs.Input[str]


   Allows to connect output_path input to the operator.

   default is {working directory}/dataProcessingDoc.html

   :returns: An Input instance for this pin.
   :rtype: input

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.utility.html_doc()
   >>> op.inputs.output_path.connect(my_output_path)
   >>> # or
   >>> op.inputs.output_path(my_output_path)

.. py:property:: exposure_level
   :type: ansys.dpf.core.inputs.Input[int]


   Allows to connect exposure_level input to the operator.

   Generate the documentation depending on exposure level : 0 (default) for public operators, 1 includes hidden operator, 2 includes private operator, 3 includes operator without specifications.

   :returns: An Input instance for this pin.
   :rtype: input

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.utility.html_doc()
   >>> op.inputs.exposure_level.connect(my_exposure_level)
   >>> # or
   >>> op.inputs.exposure_level(my_exposure_level)
