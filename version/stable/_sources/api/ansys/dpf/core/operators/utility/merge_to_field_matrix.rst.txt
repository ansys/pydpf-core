ansys.dpf.core.operators.utility.merge_to_field_matrix
======================================================

.. py:class:: ansys.dpf.core.operators.utility.merge_to_field_matrix(fields1=None, fields2=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Assembles a set of fields into a field matrix.


   :param fields1: Either a fields container, a vector of fields to merge, or fields from pin 0 to ...
   :type fields1: Field or FieldsContainer
   :param fields2: Either a fields container, a vector of fields to merge, or fields from pin 0 to ...
   :type fields2: Field or FieldsContainer

   :returns: **merged_field_matrix**
   :rtype: Field

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.utility.merge_to_field_matrix()

   >>> # Make input connections
   >>> my_fields1 = dpf.Field()
   >>> op.inputs.fields1.connect(my_fields1)
   >>> my_fields2 = dpf.Field()
   >>> op.inputs.fields2.connect(my_fields2)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.utility.merge_to_field_matrix(
   ...     fields1=my_fields1,
   ...     fields2=my_fields2,
   ... )

   >>> # Get output data
   >>> result_merged_field_matrix = op.outputs.merged_field_matrix()
