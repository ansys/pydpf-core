ansys.dpf.core.operators.utility.operator_changelog
===================================================

.. py:class:: ansys.dpf.core.operators.utility.operator_changelog(operator_name=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Return a GenericDataContainer used to instantiate the Changelog of an
   operator based on its name.


   :param operator_name: Operator internal name.
   :type operator_name: str

   :returns: **changelog_gdc** -- GenericDataContainer used to instantiate a Changelog.
   :rtype: GenericDataContainer

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.utility.operator_changelog()

   >>> # Make input connections
   >>> my_operator_name = str()
   >>> op.inputs.operator_name.connect(my_operator_name)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.utility.operator_changelog(
   ...     operator_name=my_operator_name,
   ... )

   >>> # Get output data
   >>> result_changelog_gdc = op.outputs.changelog_gdc()
