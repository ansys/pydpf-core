ansys.dpf.core.operators.mesh.split_fields
==========================================

.. py:class:: ansys.dpf.core.operators.mesh.split_fields(field_or_fields_container=None, meshes=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Split the input field or fields container based on the input mesh
   regions


   :param field_or_fields_container:
   :type field_or_fields_container: Field or FieldsContainer
   :param meshes: body meshes in the mesh controller cannot be mixed shell/solid
   :type meshes: MeshesContainer

   :returns: **fields_container**
   :rtype: FieldsContainer

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.mesh.split_fields()

   >>> # Make input connections
   >>> my_field_or_fields_container = dpf.Field()
   >>> op.inputs.field_or_fields_container.connect(my_field_or_fields_container)
   >>> my_meshes = dpf.MeshesContainer()
   >>> op.inputs.meshes.connect(my_meshes)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.mesh.split_fields(
   ...     field_or_fields_container=my_field_or_fields_container,
   ...     meshes=my_meshes,
   ... )

   >>> # Get output data
   >>> result_fields_container = op.outputs.fields_container()
