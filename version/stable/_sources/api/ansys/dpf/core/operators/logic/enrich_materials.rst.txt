ansys.dpf.core.operators.logic.enrich_materials
===============================================

.. py:class:: ansys.dpf.core.operators.logic.enrich_materials(MaterialContainer=None, streams=None, streams_mapping=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Takes a MaterialContainer and a stream and enriches the
   MaterialContainer using stream data.


   :param MaterialContainer:
   :param streams:
   :type streams: StreamsContainer or FieldsContainer
   :param streams_mapping:
   :type streams_mapping: PropertyFieldsContainer

   :returns: **MaterialContainer**
   :rtype: bool

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.logic.enrich_materials()

   >>> # Make input connections
   >>> my_MaterialContainer = dpf.()
   >>> op.inputs.MaterialContainer.connect(my_MaterialContainer)
   >>> my_streams = dpf.StreamsContainer()
   >>> op.inputs.streams.connect(my_streams)
   >>> my_streams_mapping = dpf.PropertyFieldsContainer()
   >>> op.inputs.streams_mapping.connect(my_streams_mapping)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.logic.enrich_materials(
   ...     MaterialContainer=my_MaterialContainer,
   ...     streams=my_streams,
   ...     streams_mapping=my_streams_mapping,
   ... )

   >>> # Get output data
   >>> result_MaterialContainer = op.outputs.MaterialContainer()
