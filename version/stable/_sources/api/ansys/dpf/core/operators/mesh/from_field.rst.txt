ansys.dpf.core.operators.mesh.from_field
========================================

.. py:class:: ansys.dpf.core.operators.mesh.from_field(field=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Returns the meshed region contained in the support of the mesh.


   :param field:
   :type field: Field

   :returns: **mesh**
   :rtype: MeshedRegion

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.mesh.from_field()

   >>> # Make input connections
   >>> my_field = dpf.Field()
   >>> op.inputs.field.connect(my_field)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.mesh.from_field(
   ...     field=my_field,
   ... )

   >>> # Get output data
   >>> result_mesh = op.outputs.mesh()
