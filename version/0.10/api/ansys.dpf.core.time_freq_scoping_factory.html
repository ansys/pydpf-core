time_freq_scoping_factory#

Contains functions to simplify creating time frequency scopings.

ansys.dpf.core.time_freq_scoping_factory.scoping_by_load_step(load_step: int, server=None)#

Create a specific ansys.dpf.core.Scoping for a given load step.

The returned scoping describes a specific time frequency support element for a given load step.

Parameters:

  • load_step (int) – Load step ID of the specific time frequency scoping.

  • server (DpfServer, optional) – Server with the channel connected to the remote or local instance. The default is None, in which case an attempt is made to use the global server.

Returns:

scoping – Scoping targeting one load step.

Return type:

Scoping
