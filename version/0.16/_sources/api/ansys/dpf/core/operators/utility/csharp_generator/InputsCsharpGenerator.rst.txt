





:class:`InputsCsharpGenerator`
==============================


.. py:class:: ansys.dpf.core.operators.utility.csharp_generator.InputsCsharpGenerator(op: ansys.dpf.core.dpf_operator.Operator)

   Bases: :py:obj:`ansys.dpf.core.inputs._Inputs`


   Intermediate class used to connect user inputs to
   csharp_generator operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.utility.csharp_generator()
   >>> my_dll_source_path = str()
   >>> op.inputs.dll_source_path.connect(my_dll_source_path)
   >>> my_output_path = str()
   >>> op.inputs.output_path.connect(my_output_path)
   >>> my_load_symbol = str()
   >>> op.inputs.load_symbol.connect(my_load_symbol)
   >>> my_library_key = str()
   >>> op.inputs.library_key.connect(my_library_key)
