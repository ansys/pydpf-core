Here's how you would open a result file generated by Ansys MAPDL (or another Ansys solver) and 
extract results:

.. code-block:: default

    from ansys.dpf.core import Model
    from ansys.dpf.core import examples
    model = Model(examples.find_simple_bar())
    print(model)
