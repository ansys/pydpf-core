Getting started#

DPF-Core is a Python gRPC client communicating with the Ans.Dpf.Grpc server. To use the native DPF server, you must have a local installation of Ansys 2021 R1 or later. For more information on getting a licensed copy of Ansys, visit the Ansys website.
