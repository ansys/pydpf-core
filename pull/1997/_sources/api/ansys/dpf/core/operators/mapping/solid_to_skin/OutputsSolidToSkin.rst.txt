Property detail
---------------

.. py:property:: field
   :type: ansys.dpf.core.outputs.Output[ansys.dpf.core.field.Field]


   Allows to get field output of the operator

   Field containing the mapped surface results on the skin mesh.

   The output field has:
   - **Support**: The skin mesh (pin 1)
   - **Location**: Same as input field location
   - **Scoping**: Skin mesh elements (Elemental/ElementalNodal) or skin mesh nodes (Nodal)
   - **Components**: Same structure as input field (scalar, vector, tensor, etc.)
   - **Shell layers**: Preserved if present in input field

   For Elemental and Elemental Nodal locations, skin elements whose parent solid element is not present in the input field's scoping are silently omitted. The output scoping may therefore contain fewer entries than the full skin mesh when the input field does not cover all solid elements.

   :returns: An Output instance for this pin.
   :rtype: output

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.mapping.solid_to_skin()
   >>> # Get the output from op.outputs. ...
   >>> result_field = op.outputs.field()
