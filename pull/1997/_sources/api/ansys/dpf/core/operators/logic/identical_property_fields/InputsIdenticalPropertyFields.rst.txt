





:class:`InputsIdenticalPropertyFields`
======================================


.. py:class:: ansys.dpf.core.operators.logic.identical_property_fields.InputsIdenticalPropertyFields(op: ansys.dpf.core.dpf_operator.Operator)

   Bases: :py:obj:`ansys.dpf.core.inputs._Inputs`


   Intermediate class used to connect user inputs to
   identical_property_fields operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.logic.identical_property_fields()
   >>> my_order_independent = bool()
   >>> op.inputs.order_independent.connect(my_order_independent)
   >>> my_property_fieldA = dpf.MeshedRegion()
   >>> op.inputs.property_fieldA.connect(my_property_fieldA)
   >>> my_property_fieldB = dpf.MeshedRegion()
   >>> op.inputs.property_fieldB.connect(my_property_fieldB)

Property detail

.. py:property:: order_independent
   :type: ansys.dpf.core.inputs.Input[bool]


   Allows to connect order_independent input to the operator.

   if true, the comparison between the data for each entity is order-independent (e.g. data 1,4,5 associated to id 7 is considered identical to data 4,5,1 associated to the same id). The default is false.

   :returns: An Input instance for this pin.
   :rtype: input

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.logic.identical_property_fields()
   >>> op.inputs.order_independent.connect(my_order_independent)
   >>> # or
   >>> op.inputs.order_independent(my_order_independent)
