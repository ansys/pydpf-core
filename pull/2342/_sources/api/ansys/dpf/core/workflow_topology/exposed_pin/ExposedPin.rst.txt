





:class:`ExposedPin`
===================


.. py:class:: ansys.dpf.core.workflow_topology.exposed_pin.ExposedPin(container: ansys.dpf.core.generic_data_container.GenericDataContainer)

   Bases: :py:obj:`ansys.dpf.core.custom_container_base.CustomContainerBase`


   Represents an exposed input or output pin in a workflow.

   This class provides access to the name and the associated operator, as well as its pin ID.
