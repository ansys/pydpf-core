





:class:`InputsBindSupportFc`
============================


.. py:class:: ansys.dpf.core.operators.utility.bind_support_fc.InputsBindSupportFc(op: ansys.dpf.core.dpf_operator.Operator)

   Bases: :py:obj:`ansys.dpf.core.inputs._Inputs`


   Intermediate class used to connect user inputs to
   bind_support_fc operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.utility.bind_support_fc()
   >>> my_fields_container = dpf.FieldsContainer()
   >>> op.inputs.fields_container.connect(my_fields_container)
   >>> my_support = dpf.MeshedRegion()
   >>> op.inputs.support.connect(my_support)
