





:class:`InputsInvariants`
=========================


.. py:class:: ansys.dpf.core.operators.invariant.invariants.InputsInvariants(op: ansys.dpf.core.dpf_operator.Operator)

   Bases: :py:obj:`ansys.dpf.core.inputs._Inputs`


   Intermediate class used to connect user inputs to
   invariants operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.invariant.invariants()
   >>> my_generate_principal_stress = bool()
   >>> op.inputs.generate_principal_stress.connect(my_generate_principal_stress)
   >>> my_field = dpf.Field()
   >>> op.inputs.field.connect(my_field)

Property detail
---------------

.. py:property:: generate_principal_stress
   :type: ansys.dpf.core.inputs.Input[bool]


   Allows to connect generate_principal_stress input to the operator.

   if true, generate principal stress in output pin 3/4/5 (default is false)

   :returns: An Input instance for this pin.
   :rtype: input

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.invariant.invariants()
   >>> op.inputs.generate_principal_stress.connect(my_generate_principal_stress)
   >>> # or
   >>> op.inputs.generate_principal_stress(my_generate_principal_stress)

.. py:property:: field
   :type: ansys.dpf.core.inputs.Input[ansys.dpf.core.field.Field]


   Allows to connect field input to the operator.

   :returns: An Input instance for this pin.
   :rtype: input

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.invariant.invariants()
   >>> op.inputs.field.connect(my_field)
