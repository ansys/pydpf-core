.. py:property:: normalization_type
   :type: ansys.dpf.core.inputs.Input[int]


   Allows to connect normalization_type input to the operator.

   Type of normalization applied to the residual and the norm (\\(norm\\)) :
   - 0 for absolute (default)
   - 1 for relative to the first entry
   - 2 for normalized by the max of each field of the reference field
   - 3 for normalized by the max over all fields of the reference field


   :returns: An Input instance for this pin.
   :rtype: input

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.math.compute_residual_and_error()
   >>> op.inputs.normalization_type.connect(my_normalization_type)
   >>> # or
   >>> op.inputs.normalization_type(my_normalization_type)

.. py:property:: norm_calculation_type
   :type: ansys.dpf.core.inputs.Input[int]


   Allows to connect norm_calculation_type input to the operator.

   Type for norm calculation (\\(p\\)) :

.. py:property:: field_reference
   :type: ansys.dpf.core.inputs.Input[int]


   Allows to connect field_reference input to the operator.

   Reference for the normalization step (\\(r\\)) :
   - 0 for pin 0 (default)
   - 1 for residual
   \\[ ref = \begin{cases} FC_1 &\text{ if } r = 0 \\\\ Res &\text{ if } r = 1 \end{cases} \\]


   :returns: An Input instance for this pin.
   :rtype: input

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.math.compute_residual_and_error()
   >>> op.inputs.field_reference.connect(my_field_reference)
   >>> # or
   >>> op.inputs.field_reference(my_field_reference)

.. py:property:: field_or_fields_container2
   :type: ansys.dpf.core.inputs.Input[ansys.dpf.core.field.Field | ansys.dpf.core.fields_container.FieldsContainer]


   Allows to connect field_or_fields_container2 input to the operator.

   Field or fields container of the exact same dimensions as pin 0 to be used for residual computation  (\\(FC_2\\))
   \\[ Res = \begin{cases} FC_1 - FC_2 &\text{ if } FC_2 \\\\ FC_1 &\text{ if not } FC_2 \end{cases} \\]
