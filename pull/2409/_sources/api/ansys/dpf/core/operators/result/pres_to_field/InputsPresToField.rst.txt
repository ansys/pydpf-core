





:class:`InputsPresToField`
==========================


.. py:class:: ansys.dpf.core.operators.result.pres_to_field.InputsPresToField(op: ansys.dpf.core.dpf_operator.Operator)

   Bases: :py:obj:`ansys.dpf.core.inputs._Inputs`


   Intermediate class used to connect user inputs to
   pres_to_field operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.result.pres_to_field()
   >>> my_filepath = str()
   >>> op.inputs.filepath.connect(my_filepath)
   >>> my_columns_to_read = int()
   >>> op.inputs.columns_to_read.connect(my_columns_to_read)
