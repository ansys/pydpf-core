





:class:`InputsWorkflowToPydpf`

.. py:class:: ansys.dpf.core.operators.serialization.workflow_to_pydpf.InputsWorkflowToPydpf(op: ansys.dpf.core.dpf_operator.Operator)

   Bases: :py:obj:`ansys.dpf.core.inputs._Inputs`


   Intermediate class used to connect user inputs to
   workflow_to_pydpf operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.serialization.workflow_to_pydpf()
   >>> my_workflow = dpf.Workflow()
   >>> op.inputs.workflow.connect(my_workflow)
   >>> my_output_path = str()
   >>> op.inputs.output_path.connect(my_output_path)
