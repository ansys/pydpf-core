





:class:`linearized_stress`
==========================


.. py:class:: ansys.dpf.core.operators.math.linearized_stress.linearized_stress(stress_comps=None, iComp=None, path_coords=None, rho=None, kbr=None, ksxbzro=None, out_type=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   get linearized stress


   Inputs
   ------
   stress_comps: FieldsContainer
       Stress tensor components evaluated at path points
   iComp: int
       Stress component index to linearize
   path_coords: Field
       Path point coordinates
   rho: float
       Optional geometric parameter
   kbr: int
       Option flag controlling bending/reference behavior
   ksxbzro: int
       Option flag controlling coordinate/zero-reference handling
   out_type: int
       Requested linearized stress output type.

   Outputs
   -------
   linearized_stress_results: FieldsContainer
       Selected linearized stress results for the requested output type
   linearized_stress_all_results: FieldsContainer
       All linearized stress result types computed along the path
   linearized_stress_params: FieldsContainer
       Linearization parameters and metadata

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.math.linearized_stress()

   >>> # Make input connections
   >>> my_stress_comps = dpf.FieldsContainer()
   >>> op.inputs.stress_comps.connect(my_stress_comps)
   >>> my_iComp = int()
   >>> op.inputs.iComp.connect(my_iComp)
   >>> my_path_coords = dpf.Field()
   >>> op.inputs.path_coords.connect(my_path_coords)
   >>> my_rho = float()
   >>> op.inputs.rho.connect(my_rho)
   >>> my_kbr = int()
   >>> op.inputs.kbr.connect(my_kbr)
   >>> my_ksxbzro = int()
   >>> op.inputs.ksxbzro.connect(my_ksxbzro)
   >>> my_out_type = int()
   >>> op.inputs.out_type.connect(my_out_type)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.math.linearized_stress(
   ...     stress_comps=my_stress_comps,
   ...     iComp=my_iComp,
   ...     path_coords=my_path_coords,
   ...     rho=my_rho,
   ...     kbr=my_kbr,
   ...     ksxbzro=my_ksxbzro,
   ...     out_type=my_out_type,
   ... )

   >>> # Get output data
   >>> result_linearized_stress_results = op.outputs.linearized_stress_results()
   >>> result_linearized_stress_all_results = op.outputs.linearized_stress_all_results()
   >>> result_linearized_stress_params = op.outputs.linearized_stress_params()

.. py:method:: connect(pin, inpt, pin_out=0)

   Connect an input on the operator using a pin number.

   :param pin: Number of the input pin.
   :type pin: int
   :param inpt:
   :type inpt: str, int, double, bool, list[int], list[float], Field, FieldsContainer, Scoping,
   :param ScopingsContainer: Operator, os.PathLike Object to connect to.
   :param MeshedRegion: Operator, os.PathLike Object to connect to.
   :param MeshesContainer: Operator, os.PathLike Object to connect to.
   :param DataSources: Operator, os.PathLike Object to connect to.
   :param CyclicSupport: Operator, os.PathLike Object to connect to.
   :param dict: Operator, os.PathLike Object to connect to.
   :param Outputs: Operator, os.PathLike Object to connect to.
   :param pin_out: If the input is an operator, the output pin of the input operator. The default is ``0``.
   :type pin_out: int, optional

   .. rubric:: Examples

   Compute the minimum of displacement by chaining the ``"U"`` and ``"min_max_fc"`` operators.

   >>> from ansys.dpf import core as dpf
   >>> from ansys.dpf.core import examples
   >>> data_src = dpf.DataSources(examples.find_multishells_rst())
   >>> disp_op = dpf.operators.result.displacement()
   >>> disp_op.inputs.data_sources(data_src)
   >>> max_fc_op = dpf.operators.min_max.min_max_fc()
   >>> max_fc_op.inputs.connect(disp_op.outputs)
   >>> max_field = max_fc_op.outputs.field_max()
   >>> max_field.data
   DPFArray([[0.59428386, 0.00201751, 0.0006032 ]]...
