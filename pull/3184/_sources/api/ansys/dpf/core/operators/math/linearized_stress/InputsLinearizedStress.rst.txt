





:class:`InputsLinearizedStress`
===============================


.. py:class:: ansys.dpf.core.operators.math.linearized_stress.InputsLinearizedStress(op: ansys.dpf.core.dpf_operator.Operator)

   Bases: :py:obj:`ansys.dpf.core.inputs._Inputs`


   Intermediate class used to connect user inputs to
   linearized_stress operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.math.linearized_stress()
   >>> my_stress_comps = dpf.FieldsContainer()
   >>> op.inputs.stress_comps.connect(my_stress_comps)
   >>> my_iComp = int()
   >>> op.inputs.iComp.connect(my_iComp)
   >>> my_path_coords = dpf.Field()
   >>> op.inputs.path_coords.connect(my_path_coords)
   >>> my_rho = float()
   >>> op.inputs.rho.connect(my_rho)
   >>> my_kbr = int()
   >>> op.inputs.kbr.connect(my_kbr)
   >>> my_ksxbzro = int()
   >>> op.inputs.ksxbzro.connect(my_ksxbzro)
   >>> my_out_type = int()
   >>> op.inputs.out_type.connect(my_out_type)
