





:class:`InputsPropertyField`
============================


.. py:class:: ansys.dpf.core.operators.utility.property_field.InputsPropertyField(op: ansys.dpf.core.dpf_operator.Operator)

   Intermediate class used to connect user inputs to
   property_field operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.utility.property_field()

   >>> op.inputs.input.connect(my_input)
